# Виртуальный музей индустриальной истории Красногвардейского района

Современная цифровая платформа, посвящённая индустриальной истории, архивам и промышленному наследию Красногвардейского района Санкт-Петербурга.

## О проекте

«Виртуальный музей индустриальной истории Красногвардейского района» — это digital museum / historical archive platform, в которой промышленная история рассматривается как часть городской памяти и культурной идентичности территории.

Проект объединяет в едином визуальном пространстве:

- архивные материалы;
- исторические и современные фотографии;
- предприятия и производственные площадки;
- инфраструктурные объекты;
- тематические исследования;
- хронологию индустриального развития района.

## Основные возможности сайта

- каталог объектов промышленного наследия;
- визуальная архивная коллекция;
- тематические исследования и подборки;
- историческая хронология ключевых этапов;
- карточки объектов с дополнительными материалами;
- адаптивный интерфейс для desktop / tablet / mobile;
- современный editorial-style UI с музейной подачей.

## Визуальная концепция

Проект строится на сочетании:

- dark industrial style;
- digital archive aesthetics;
- museum-inspired UI;
- editorial layout;
- атмосферы индустриального Петербурга.

В основе — аккуратная сеточная композиция, визуальные акценты и работа с архивным ощущением без перегруза эффектами.

## Структура проекта

Актуальная структура репозитория:

```text
.
├── indastrial.html
├── README.md
├── ТЕХНИЧЕСКОЕ ЗАДАНИЕ.docx
└── assets/
    └── images/
        ├── archive/         # изображения для блока архивной коллекции
        ├── collections/     # изображения для тематических исследований
        ├── hero/            # hero-изображения
        ├── objects/         # изображения карточек объектов и галерей
        ├── timeline/        # изображения исторической хронологии
        ├── real/            # реальные изображения, добавленные в проект
        ├── placeholders/    # резервные заглушки
        ├── seed/            # служебные исходники для тестового наполнения
        └── sources.csv      # таблица источников изображений
```

## Текущий статус проекта

Проект находится на стадии **MVP / prototype**.

Сейчас ведётся:

- развитие визуальной структуры и композиции блоков;
- уточнение музейной editorial-подачи;
- замена временных решений на полноценное контентное наполнение.

Следующий этап — дальнейшее расширение архива материалов и каталога объектов.

## Возможное развитие

- расширение каталога промышленных объектов;
- углубление цифрового архива документов и фотоисточников;
- новые тематические исследования и спецподборки;
- интеграция мультимедийных форматов (аудио, видео, интерактивные материалы);
- развитие проекта как устойчивой музейной digital-платформы.

## Запуск проекта

### Вариант 1 — быстрый локальный просмотр

1. Склонировать или скачать репозиторий.
2. Открыть файл `indastrial.html` в браузере.

### Вариант 2 — через Live Server (рекомендуется)

1. Открыть папку проекта в VS Code.
2. Запустить `Live Server` для файла `indastrial.html`.

Это удобно для корректной проверки путей к изображениям и быстрых визуальных итераций.

## GitHub Pages

Проект можно опубликовать через GitHub Pages как публичный digital archive website.

Для этого достаточно разместить актуальную версию HTML и папку `assets` в основной ветке репозитория и включить GitHub Pages в настройках.

## Контакты / авторство

Проект разработан как культурно-цифровая инициатива, посвящённая городской индустриальной памяти Красногвардейского района.

По вопросам развития, контента и сотрудничества можно использовать контакты, указанные в интерфейсе сайта.
